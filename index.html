what is html

"HTML is the most basic building block of the Web."

HTML is a markup language composed of elements which wrap text content, defining its structure and behaviour.

Some basic HTML elements:

  • <!doctype html>: a required preamble that declares that this document is HTML5
  • <html>: the root element of an HTML page
  • <head>: encloses meta information about the page like <title>, the page tab's title
  • <body>: defines the document's body, is the container for all visible contents
  • <h1> to <h6>: define headings of various sizes
  • <p>: defines a paragraph
  • <a href="">: defines a hyperlink, short for anchor
  • <img src="" alt="alt text" />: embeds an image located at src with alternative text
  • also: an HTML comment is enclosed with <!-- and -->

html is a little plain! enter css

CSS (cascading style sheets) is a style sheet language used to style HTML elements. This happens outside of the actual html file, in a .css file.

To apply it, it must be linked in the HTML head.

A style sheet is composed of rulesets, each of which apply to an element. The element is identified with the selector, which is followed by curly braces which enclose the declaration. Within the declaration are properties being set by property values, separated with semicolons.
